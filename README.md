# NEKOFETCH
A simple web scrapper to grab cat girl images from different images aggregators

# OPERATIONAL
- Bash scipt
- Python script

# TO-DO

# HOW TO
## BASH SCRIPT
`./script [option]`
```
h : help
s : search for images
f : display images on feh
c : clear curently downloaded images
```
